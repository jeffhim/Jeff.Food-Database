安全規則設定
🔒 鎖定只限你使用：

1. 到 Firebase Console → Firestore → 規則
2. 登入後複製上方的 UID
3. 將規則改為：
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /nutrition/{docId} {
      allow read, write: if request.auth != null
        && request.auth.uid == '你的UID';
    }
  }
}
4. 將 '你的UID' 替換為你的 UID
5. 點「發布」
